# 🍴 RapidRecipe

Aplicación móvil de recetas rápidas, saludables y económicas.

## Descripción
RapidRecipe ayuda a estudiantes y profesionales a encontrar
recetas fáciles adaptadas a sus ingredientes y tiempo disponible.
